# ☁️ 초보자를 위한 클라우드 배포 가이드 (GitHub & Streamlit Cloud)

이 가이드는 컴퓨터를 꺼도 24시간 뉴스 대시보드에 접속할 수 있도록 인터넷에 올리는 방법을 아주 쉽게 설명합니다.

복잡한 명령어 대신 **마우스 클릭과 드래그**로만 할 수 있는 방법을 알려드릴게요!

---

## 1단계: GitHub 가입 및 저장소 만들기
**GitHub(깃허브)**는 코드를 저장하는 인터넷 파일 보관소입니다.

1.  [GitHub 가입하기](https://github.com/join) (이미 계정이 있으면 로그인)
2.  로그인 후, 오른쪽 상단의 **`+`** 버튼을 누르고 **`New repository`**를 선택합니다.
3.  **Repository name** 칸에 영어로 이름을 짓습니다. (예: `financial-news-dashboard`)
4.  **Public** (공개)이 선택되어 있는지 확인합니다.
5.  맨 아래 초록색 **`Create repository`** 버튼을 클릭합니다.

---

## 2단계: 파일 업로드하기 (가장 중요!)
명령어를 치는 화면이 나와도 당황하지 마세요. 우리는 **웹사이트 업로드** 기능을 쓸 겁니다.

1.  방금 만든 화면에서 파란색 글씨로 된 **`uploading an existing file`** 링크를 찾아서 클릭합니다.
    *   (못 찾겠으면 주소창 뒤에 `/upload`를 붙여보세요. 예: `github.com/내아이디/내프로젝트/upload`)
2.  **내 컴퓨터의 프로젝트 폴더**를 엽니다.
    *   경로: `c:\Users\pkt14\OneDrive\바탕 화면\자동화 에이전트들\financial-news-dashboard`
3.  폴더 안의 **모든 파일**을 선택해서 GitHub 웹페이지의 박스 안으로 **드래그 앤 드롭** 합니다.
    *   (주의: `.git` 폴더나 `venv` 폴더가 보이면 그건 빼고 올리세요. 나머지는 다 올려도 됩니다.)
4.  파일 목록이 주르륵 뜨며 업로드가 될 때까지 기다립니다.
5.  모두 완료되면, 아래쪽 **Commit changes** (녹색 버튼)를 클릭합니다.

이제 코드가 인터넷에 올라갔습니다! 🎉

---

## 3단계: Streamlit Cloud 연결하기
이제 코드를 실제로 실행시켜줄 서버를 연결합니다.

1.  [Streamlit Community Cloud](https://share.streamlit.io/) 에 접속합니다.
2.  **`Continue with GitHub`** 버튼을 눌러 로그인합니다.
3.  오른쪽 상단의 파란색 **`New app`** 버튼을 클릭합니다.
4.  **`Use existing repo`**를 선택합니다.
5.  **Repository** 칸을 클릭하면 방금 만든 `financial-news-dashboard`가 보일 겁니다. 선택하세요.
6.  **Main file path** 칸에 `app.py`라고 적혀있는지 확인합니다.
7.  **`Deploy!`** 버튼을 힘차게 클릭합니다. 🎈

---

## 4단계: 완료 및 접속
1.  화면 하단에 "Baking your app..." 하면서 요리하는 애니메이션이 나옵니다. (설치 중이라는 뜻)
2.  약 2~3분 정도 기다리면 대시보드 화면이 뜹니다!
3.  이제 브라우저 주소창에 있는 링크(예: `https://financial-news-dashboard-xxxx.streamlit.app`)를 복사해서 카톡으로 보내두세요.
4.  **핸드폰**에서 그 링크를 누르면 언제 어디서든 뉴스를 볼 수 있습니다!

---

### 💡 팁
*   **파일 수정**: 코드를 고치고 싶으면 내 컴퓨터에서 고친 뒤, 다시 2단계 방법으로 해당 파일만 드래그해서 덮어씌우면 됩니다. (Commit changes 누르면 자동 반영)
*   **무료**: 이 모든 과정은 무료입니다.

## 5단계: 웹사이트 업데이트 방법 (수정사항 반영)

**지금 바로 '채용 정보'와 '새로운 디자인'을 적용하고 싶으신가요?**
아래 과정을 따라하시면 1분 안에 끝납니다!

1.  **GitHub 저장소 열기**:
    *   인터넷에서 내 GitHub 저장소 (`financial-news-dashboard`) 페이지로 들어갑니다.
2.  **파일 업로드 메뉴**:
    *   파일 목록 위에 있는 **`Add file`** 버튼을 누르고 -> **`Upload files`**를 클릭합니다.
3.  **파일 드래그 앤 드롭**:
    *   내 컴퓨터의 프로젝트 폴더(`c:\Users\pkt14\OneDrive\바탕 화면\자동화 에이전트들\financial-news-dashboard`)를 엽니다.
    *   **`app.py`**, **`company_data.py`**, **`news_fetcher.py`**, **`report_generator.py`** 이 4개 파일을 선택해서 인터넷 창에 끌어다 놓습니다.
    *   (그냥 폴더에 있는 파일 다 끌어다 놓으셔도 됩니다. 알아서 바뀐 것만 처리해줍니다!)
4.  **저장 버튼 클릭**:

5.  **기다리기**:

    *   약 1~2분 뒤에 핸드폰에서 **새로고침** 하시면 바뀐 화면이 나옵니다!
