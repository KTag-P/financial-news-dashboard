# 🏦 캐피탈사 채용 대비 Dashboard

> IBK캐피탈 · 산은캐피탈 종합 분석 대시보드 — 채용 준비를 위한 원스톱 정보 플랫폼

## 📋 프로젝트 개요

금융권(캐피탈사) 취업 준비생을 위한 종합 분석 대시보드입니다. IBK캐피탈과 산은캐피탈의 기업 정보, 재무 지표, 채용 정보, 뉴스 아카이브 등을 한 곳에서 확인할 수 있습니다.

## ✅ 구현 완료 기능

### 1. 기업 개요 탭 (`/index.html`)
- **핵심 통계 카드** (4개): IBK/KDB 총자산, 순이익, 연체율 등 핵심 KPI 한눈에 확인
- **실시간 시장 지표** (10개): KOSPI, KOSDAQ, 환율, 미국채, NASDAQ, 금/은 가격 등
- **IBK캐피탈 vs 산은캐피탈 비교 패널**:
  - 🔑 2026 경영 전략 (Key Issues)
  - 💼 주요 사업 (Business Areas) — 아코디언 펼치기
  - 📊 재무 지표 테이블 (2023~2025E)
  - 📜 연혁 타임라인
  - 🧑‍🤝‍🧑 채용 정보 (예정/완료 구분)
  - 📋 성공/실패 사례 분석

### 2. 재무 비교 탭
- **당기순이익 비교 차트** (Bar Chart)
- **ROE 추이 차트** (Line Chart)
- **총자산 비교 차트** (Bar Chart)
- **2024년 실적 핵심 비교 테이블** (7개 항목)

### 3. 뉴스 아카이브 (4개 탭)
- **IBK캐피탈 뉴스** / **산은캐피탈 뉴스** / **캐피탈 업황** / **거시경제**
- 키워드 검색 (제목 + 본문)
- 연도/월 필터링
- AI 요약 보기 + 원문 전체 보기
- 페이지네이션

### 4. 디자인 & UX
- 🌓 **다크모드** 지원 (시스템 설정 연동 + 수동 전환)
- ✨ **스크롤 애니메이션** (Intersection Observer 기반 fade-in)
- 🎨 **그라디언트 + 글래스모피즘** 디자인 시스템
- 📱 **반응형** 레이아웃 (모바일/태블릿/데스크톱)
- ⬆️ 스크롤 투 탑 버튼
- 부드러운 탭 전환 애니메이션

## 📁 프로젝트 구조

```
/
├── index.html              # 메인 페이지
├── css/
│   └── style.css           # 전체 스타일시트 (다크모드 포함)
├── js/
│   ├── data.js             # 기업 데이터, 재무 정보, 사례 데이터
│   └── app.js              # 앱 로직 (탭, 차트, 뉴스, 다크모드)
├── reference/
│   ├── news_history.json   # 뉴스 아카이브 데이터 (3.3MB)
│   ├── app.py              # 원본 Streamlit 코드 (참조용)
│   ├── company_data.py     # 원본 Python 데이터 (참조용)
│   ├── report_generator.py # 원본 보고서 생성기 (참조용)
│   ├── recruitment_fetcher.py
│   └── recruitment_data.json
└── README.md
```

## 🔗 진입 URI

| 경로 | 설명 |
|---|---|

| `#panel-charts` | 재무 비교 탭 |

## 🔧 기술 스택

- **HTML5** + **CSS3** (Custom Properties, Grid, Flexbox, Animations)
- **Vanilla JavaScript** (ES6+, Intersection Observer, LocalStorage)
- **Chart.js 4.4** — 재무 차트 시각화
- **Pretendard** — 한글 웹폰트
- **Font Awesome 6.4** — 아이콘
